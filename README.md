# Hi, I'm Julian

PhD researcher in Computer Vision and Computer Graphics at the University of Bonn, with a focus on dynamic 3D/4D reconstruction, neural representations, Gaussian Splatting, and human avatar reconstruction.

Research is centered on efficient and robust methods for reconstructing geometry, appearance, and motion from real-world capture data, with applications in virtual humans, AR/VR, telepresence, and digital content creation.

## Research Interests

* Dynamic 3D/4D reconstruction
* Neural fields and learned scene representations
* 3D/4D Gaussian Splatting and Gaussian avatars
* Human body and avatar reconstruction
* Multi-view geometry and camera-based capture
* Efficient optimization and deep learning for graphics/vision

## Selected Work

### Neu-PiG — Neural Preconditioned Grids for Fast Dynamic Surface Reconstruction

**CVPR 2026**
[Paper](https://arxiv.org/html/2602.22212v2) · [Code](https://github.com/vc-bonn/neu-pig.git)

Fast and temporally stable dynamic surface reconstruction from unstructured point clouds using neural preconditioned latent grids.

### PiG-Avatar — Hierarchical Neural-Field-Guided Gaussian Avatars

**arXiv preprint, 2026**
[Paper](https://arxiv.org/html/2605.20185v2)

Template-decoupled Gaussian avatar reconstruction using volumetric canonical anchors and neural-field-guided optimization.

### Preconditioned Deformation Grids

**Computer Graphics Forum / Pacific Graphics 2025**
[Paper](https://arxiv.org/html/2509.18097v1) · [Code](https://github.com/vc-bonn/preconditioned-deformation-grids)

A stable optimization-based approach for long-sequence dynamic geometry reconstruction.

### ROSA — Reconstructing Object Shape and Appearance Textures by Adaptive Detail Transfer

**WACV 2025**
[Paper](https://arxiv.org/pdf/2501.18595) · [Code](https://github.com/vc-bonn/rosa)

Static object reconstruction method combining geometry and appearance transfer for high-quality shape and texture recovery.
